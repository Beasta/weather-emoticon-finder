# unicode-weather

Retrieve weather emoticon for various weather conditions.

## Installation

```
npm install weather-emoticon-finder
```

## Usage
```js

var WeatherEmoticonFinder = require('weather-emoticon-finder');
console.log(WeatherEmoticonFinder().get('clear'));
// ☀️
```
## Running the tests

```
npm test
```
